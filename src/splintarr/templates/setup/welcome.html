Welcome to Splintarr

Intelligent backlog search automation for Sonarr and Radarr

What is Splintarr?

Splintarr automatically searches your Sonarr and Radarr backlog for unwatched or missing content, intelligently prioritizing items based on your preferences and viewing habits.

Key Features

  • Automated backlog searching with customizable strategies
  • Sonarr TV show support (Radarr coming in a future release)
  • Search intelligence with adaptive prioritization and cooldowns
  • Season pack detection with individual episode fallback
  • Prowlarr integration for indexer-aware rate limiting
  • Discord notifications for search activity
  • Content exclusion lists
  • Encrypted database with secure API key storage

Getting Started

This setup wizard will guide you through:

  1. Creating an admin account
  2. Connecting your Sonarr instance
  3. Configuring Discord notifications (optional)
  4. Connecting Prowlarr (optional)
